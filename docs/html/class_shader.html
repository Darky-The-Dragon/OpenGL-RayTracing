Detailed Description

Minimal RAII wrapper for an OpenGL shader program.

This class handles compilation, linking, binding, and uniform management for GLSL vertex/fragment programs. It is intentionally lightweight but provides convenience helpers for setting common uniform types.

The class is:

  • non-copyable (avoids double deletion of GL programs)
  • movable (safe transfer of ownership)

The constructor takes file paths and builds a complete shader program.
