Detailed Description

Minimal geometry buffer storing world-space position and normal.

The GBuffer is sized to match the framebuffer resolution and provides per-pixel world-space attributes required for temporal reprojection, denoising, and certain debug visualizations. It stores:

  • posTex : RGB16F world-space position
  • nrmTex : RGB16F world-space normal

Both textures are allocated as floating-point formats to preserve enough precision for ray tracing and shading calculations.
