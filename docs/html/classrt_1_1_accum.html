Detailed Description

Manages the temporal accumulation buffer used by the path tracer.

This class encapsulates all GPU resources required for progressive accumulation (ping-pong textures), motion-vector storage, and MRT output. It handles creation, resizing, clearing, and binding of the underlying framebuffer object.

The accumulation buffer stores:

  • linear HDR color (RGBA16F)
  • screen-space motion vectors (RG16F)

The ping-pong scheme alternates between two color textures each frame. One texture is written to (current frame), while the other is sampled from (previous accumulated result). The class keeps track of both the active index and the current accumulation frame count.

Creates or recreates all accumulation textures.

Called on window resize or initial startup. This function allocates:

  • two RGBA16F accumulation textures
  • one RG16F motion vector texture and attaches them to the FBO. Previous resources are deleted.
