Detailed Description

Stores camera matrices and temporal information needed per frame.

The renderer uses this struct to compute motion vectors, accumulate temporal history, and determine whether reprojection is valid between frames. Both current and previous view-projection matrices are tracked, along with camera positions and the current pixel jitter value used for TAA-style sampling.
