Detailed Description

Node structure for a median-split BVH.

Internal nodes store a bounding box and indices of their children. Leaf nodes store the starting triangle index and the number of triangles.

Conventions:

  • left/right = child indices or -1 if none
  • first/count are valid only for leaf nodes
