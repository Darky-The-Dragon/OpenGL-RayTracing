Detailed Description

Stores all per-frame and edge-triggered input information.

This struct acts as the central input buffer for the entire engine. It tracks runtime-tunable parameters (SPP, exposure), key-edge transitions (e.g., toggling ray mode or BVH), mouse deltas, and state required for pointer-lock / camera control.

The separation between "edge-triggered" (press once) and "runtime-tunable" (continuous) fields helps the main loop keep rendering logic deterministic and avoids mixing GLFW queries inside the renderer.
