void io::attach_callbacks ( GLFWwindow * window)

Attaches GLFW mouse and scroll callbacks used for camera control.

These callbacks only register raw input data (mouse deltas and scroll offsets). Interpretation—such as updating camera orientation or changing FOV—is handled later inside io::update().

Parameters
windowGLFW window to attach event callbacks to.

◆ update()

bool io::update ( InputState & s,
GLFWwindow * win )

Processes GLFW input and updates the InputState for this frame.

This function handles:

  • key edge detection (toggles, mode switches, SPP changes)
  • ESC quit request
  • FOV changes from scroll input
  • pointer lock transitions

The main loop uses the returned boolean to determine whether the accumulation buffer must be reset (e.g., camera moved, SPP changed).

Parameters
sReference to the input state that will be updated.
winPointer to the GLFW window used for polling.
Returns
True if any input event requires re-render accumulation reset.
