glm::mat4 defaultBvhTransform ( )
inline

Default model transform applied to BVH models.

This transform mirrors the placement used for the rasterized bunny model to ensure visual consistency across rendering modes (raster vs ray tracing). It is kept as a small helper since BVH models do not have a fixed origin.

Returns
A model matrix with translation and uniform scale.
