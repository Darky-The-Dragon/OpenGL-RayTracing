Detailed Description

Triangle representation used during BVH construction.

The triangle is stored in a format optimized for ray intersection:

  • v0 is the first vertex position
  • e1 = v1 - v0
  • e2 = v2 - v0

This layout allows computing rayâ€“triangle intersections with only dot products, avoiding extra recomputation of edges during traversal.
