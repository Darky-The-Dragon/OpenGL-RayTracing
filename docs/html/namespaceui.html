Draws the application's control panel.

This UI exposes:

  • Material and lighting parameters (RenderParams)
  • Toggling between raster and ray tracing modes
  • BVH usage toggle and BVH model picker
  • Motion-vector visualization options
  • Debug information from InputState and FrameState
Parameters
paramsRender parameters to modify.
frameCurrent frame state for debug visualization.
inputInput state (read-only).
rayModeToggle between raster and ray/path tracing.
useBVHToggle BVH acceleration structure.
showMotionToggle motion-vector debug mode.
bvhPickerUI state for BVH model selection.
envPickerUI state for environment map selection.
