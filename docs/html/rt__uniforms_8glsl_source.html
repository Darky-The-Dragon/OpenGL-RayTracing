5/*
6 rt_uniforms.glsl – Shared uniform block for ray tracing & present passes
7
8 This file centralizes all GLSL uniforms used by the ray tracing pipeline
9 (primary ray generation, analytic scene, BVH traversal, GI/AO, materials,
10 environment lighting, and TAA/SVGF).
11
12 The idea is to keep CPU ↔ GPU bindings in one place so:
13 - The C++ side (RenderParams + AppState) has a single source of truth
14 for parameter names and semantics.
15 - All the GLSL helper modules (rt_common, rt_lighting, rt_scene_analytic,
16 rt_bvh, rt_taa, etc.) can include this and agree on the same layout.
17
18 Most of these uniforms are directly backed by RenderParams on the C++ side
19 and are updated every frame via the UI / ImGui controls.
20*/
21
